## 📖 Tentang Proyek

**AR Avometer Learning Simulator** merupakan aplikasi edukasi berbasis **Unity** yang menggabungkan teknologi **Augmented Reality (AR)** dengan komunikasi data secara real-time.

Proyek ini memungkinkan pengguna mempelajari dan mensimulasikan penggunaan **avometer (multimeter)** langsung melalui smartphone. Pengguna dapat memasukkan nilai pengukuran melalui aplikasi berbasis **Kodular**, mengirim data melalui **Firebase**, lalu hasilnya langsung ditampilkan pada avometer virtual di aplikasi Unity AR.

Dengan aplikasi ini, proses belajar tidak hanya teori, tetapi juga praktik simulasi secara interaktif.

## ✨ Fitur Utama

- ✅ Visualisasi **Avometer 3D berbasis AR**
- ✅ Media pembelajaran interaktif menggunakan kamera smartphone
- ✅ Input nilai dari aplikasi eksternal
- ✅ Koneksi real-time menggunakan **Firebase Realtime Database**
- ✅ Mode simulasi pengukuran:
  - 🔸 Dioda
  - 🔸 Resistansi (Ohm)
  - 🔸 Tegangan AC
  - 🔸 Tegangan DC
- ✅ Tersedia panduan penggunaan aplikasi
- ✅ Pengalaman belajar modern dan imersif

## 🧠 Latar Belakang

Pembelajaran alat ukur listrik sering kali terbatas pada teori atau penggunaan alat fisik yang jumlahnya terbatas.

Proyek ini dibuat untuk memberikan:

- Pengalaman belajar yang lebih menarik
- Simulasi aman tanpa risiko listrik nyata
- Mudah diakses melalui smartphone
- Visualisasi lebih jelas untuk pemula
- Mendukung belajar mandiri maupun di kelas
